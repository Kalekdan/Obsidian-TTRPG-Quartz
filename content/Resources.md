This is a collection of resources used to assist the worldbuilding of The Isles of Fate.

##  Map Creation 
###  World Maps 

####  Azgaar's Fantasy Map Generator 
[Azgaar's Fantasy Map Generator](https://azgaar.github.io/Fantasy-Map-Generator/) is a useful tool for randomly generating world maps, but also has some options to refine and edit the maps. This was the tool initially used to generate the world map. To edit, import the existing map (![[map_gen_save.map]]) and edit from there. Replace the version of the map stored here to keep up to date

###  Settlement Maps 
####  Watabou's City Generator 
[Watabou's City Generator](https://watabou.github.io/city-generator) is a tool to randomly generate smaller scale cities and settlements. To populate a settlement with people, places and plots [Eigengrau's Generator](https://eigengrausgenerator.com/) is super powerful.

##  Character Creation 
Character creation can be done however the players like, however [D&D Beyond](https://www.dndbeyond.com) is encouraged due to its friendly and very interactive interface. It is useful to also create a copy of the character in [Roll20](https://www.roll20.net/welcome) as that is where we will play the majority of our sessions, or alternatively, use [Beyond20](https://chrome.google.com/webstore/detail/beyond-20/gnblbpbepfbfmoobegdogkglpbhcjofh?hl=en) to link the two environments together.

[HeroForge](https://www.heroforge.com/) is a great site for making 3D models of your characters, with loads of options for customisation, and can also order them as miniatures (not cheap though). If you create a free account you can also save the models.

###  NPC Generation 
To quickly generate a random NPC with a brief plot point and a portrait use [DM Heroes.](http://www.dmheroes.com/)

##  Handout Generation 
[Perchance](https://perchance.org/newspaper-column-creator) can be used to create easy newspaper columns.

[Scriptalizer](http://www.scriptalizer.co.uk/scriptalizer.aspx) is really good for generating handwritten type text, and can be screenshotted to be used as handouts for notes etc.
[10015.io](https://10015.io/tools/text-to-handwriting-converter) also generates handwritten text in a cleaner format.
##  Music

[Roll20](https://www.roll20.net/welcome) is the primary tool used for hosting virtual sessions.

##  Cheat Sheets/Resource collections 
When in combat, [Combat QuickRef](https://crobi.github.io/dnd5e-quickref/preview/quickref.html) is a really good resource as a quick reminder of what you can do in a given turn.
For key points that I want to reference, I've also got a personal [[cheat sheet]] that I use to log specific rules important to me.

[5eTools](https://5e.tools), and in particular [the bestiary](https://5e.tools/bestiary.html) is an invaluable resource for references for traps/conditions etc, as well as tokens for monsters.

[Aidedd](https://www.aidedd.org/dnd-filters/monsters.php) can also be useful for putting loads of monster sheets/spells on a single page. It also has [Unearthed Arcana](https://www.aidedd.org/en/unearthed-arcana/) resources, all collected together here.

[The Thieves Guild](https://www.thievesguild.cc) has all sorts of resouces, but particularly useful are the shop inventory lists.

![[d_d_5e_-_dms_cheat_sheet.pdf]] is about 10 pages of super useful information to have on hand (name lists, costs, encounters etc). Worth printing and having available when running a session.
